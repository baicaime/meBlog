卸载腾讯云的云监控组件 sgagent、barad_agent、YDService

2025.12.31 14:57 腾讯云 qcloud

相关云监控的进程分别是sgagent,barad_agent和YDService

删除云监控插件

/usr/local/qcloud/stargate/admin/uninstall.sh
/usr/local/qcloud/YunJing/uninst.sh
/usr/local/qcloud/monitor/barad/admin/uninstall.sh

删除自动化助手

sudo wget -qO - https://tat-1258344699.cos.accelerate.myqcloud.com/tat_agent/uninstall.sh | sudo sh

或

systemctl stop tat_agent
systemctl disable tat_agent
rm -f /etc/systemd/system/tat_agent.service
rm -rf /usr/local/qcloud

检查 agent 进程是否还在

ps -A | grep agent

如果执行后没有任何显示，说明已经清理干净

删除自启

进入服务器 /etc/rc.d/rc.local 目录，删除所有/usr/local/qcloud/路径的信息。

官方文档

云监控：https://cloud.tencent.com/document/product/248/53584
YDService ：https://cloud.tencent.com/document/product/296/9928
自主化助手：https://cloud.tencent.com/document/product/1340/50822
