使用 fdisk 或 parted 把第二块硬盘分区并挂载

2026.07.01 19:37 技术分享 linux fdisk parted

系统环境： Debian13

使用 fdisk 分区

先确认：

fdisk -l

如果能正常输出，就可以用 fdisk。

进入分区工具：

fdisk /dev/vdb

依次输入：

g    # 创建新的 GPT 分区表
n    # 新建分区
<回车>  # 分区号，默认 1
<回车>  # 起始位置，默认
<回车>  # 结束位置，默认（使用整个磁盘）
w    # 保存退出

完成后会生成：

/dev/vdb1

查看确认：

lsblk

应该能看到类似：

vdb     500G
└─vdb1  500G

使用 parted 分区

parted /dev/vdb --script mklabel gpt
parted /dev/vdb --script mkpart primary ext4 0% 100%

得到：

/dev/vdb1

然后格式化

例如 ext4：

mkfs.ext4 /dev/vdb1

挂载

例如：

mkdir -p /data
mount /dev/vdb1 /data

查看：

df -h

应该能看到：

/dev/vdb1    500G   ...   /data

设置开机自动挂载

查看 UUID：

blkid /dev/vdb1

例如：

UUID="12345678-xxxx-xxxx"

编辑：

nano /etc/fstab

加入：

UUID=12345678-xxxx-xxxx  /data  ext4  defaults,nofail  0  2

然后测试：

mount -a

没有报错即可。
